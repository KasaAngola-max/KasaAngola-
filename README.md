## 🚀 Como publicar no Netlify (GRÁTIS)

### Passo 1 — Criar conta no Netlify
1. Acesse: https://netlify.com
2. Clique em **Sign up** → Entre com Google ou GitHub (gratuito)

### Passo 2 — Fazer upload do projeto
1. No painel do Netlify, clique em **"Add new site"**
2. Escolha **"Deploy manually"**
3. Arraste a pasta **`connectme`** para a área indicada

### Passo 3 — Pronto! 🎉
- O Netlify vai gerar um link como: `https://connectme-abc123.netlify.app`
- Compartilhe esse link com qualquer pessoa
- Funciona em celular, tablet e computador

---

## 📱 Instalar como App no Celular

### Android (Chrome):
1. Abra o link no Chrome
2. Toque nos **3 pontinhos** no canto
3. Toque em **"Adicionar à tela inicial"**
4. Pronto! Aparece como app no celular

### iPhone (Safari):
1. Abra o link no Safari
2. Toque no botão de **compartilhar** (quadrado com seta)
3. Toque em **"Adicionar à Tela de Início"**
4. Pronto!
